Name: Mishal Fatima

## Assigned Hierarchy
Person → Student → GraduateStudent  

## Description
This project demonstrates multilevel inheritance in Java.  
It includes three classes where Student inherits Person and GraduateStudent inherits Student.  

Concepts used:
- Inheritance
- Constructor chaining using super
- Method overriding
- Use of this keyword

The Test class shows how all classes work and interact with each other.
